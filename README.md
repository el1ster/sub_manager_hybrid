# Менеджер підписок Hybrid (sub_manager_hybrid)

Гібридний додаток для управління підписками (Desktop + Telegram Bot) з акцентом на безпеку та зручність обліку в гривнях (UAH).

## Основні характеристики
- **Гібридна платформа:** Керування через десктопний інтерфейс (PyQt6) та Telegram-бот (aiogram).
- **Безпека:** Шифрування даних за стандартом AES-256 (cryptography).
- **Валютний облік:** Основна валюта — UAH.
- **Аналітика:** Візуалізація витрат за допомогою pyqtgraph та обробка даних через pandas.

## Технологічний стек
- Python 3.13+
- PySide6 (Desktop UI)
- aiogram (Telegram Bot API)
- SQLAlchemy (ORM для роботи з SQLite)
- cryptography (шифрування AES-256)
- pandas & pyqtgraph (аналітика та графіки)

## Структура проекту
- `src/core/`: Спільна бізнес-логіка, моделі даних (SQLAlchemy) та безпека (AES-256).
- `src/database/`: Керування сесіями БД та репозиторії.
- `src/ui/`: Десктопний інтерфейс на PySide6.
- `src/bot/`: Telegram-бот на aiogram 3.x.
- `src/assets/`: Статичні ресурси.

## Встановлення
1. Клонуйте репозиторій.
2. Створіть віртуальне середовище: `python -m venv .venv`
3. Встановіть залежності: `pip install -r requirements.txt`
